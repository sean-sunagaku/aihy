# ADR-006: コメント規約の導入

- Date: 2026-04-23
- Status: Accepted

## 背景
AI が判断根拠を残す場所が必要だが、本文に混ぜるとペルソナの主観と区別できない。人間向けテンプレヒントも同様。

## 決定
HTML コメントで2種類を分離:
- `<!-- ヒント -->`: 人間向けガイド（テンプレに埋め込み、書き終えたら削除可）
- `<!-- Claude注: ... -->`: AI 自身の判断根拠・迷い・代替案（次サイクル Read 時に文脈として参照）

## 影響
- `problem.md` テンプレの各セクションにヒントを埋め込み
- `session.md` / `feedback.md` で AI が判断を積み上げられる
- HTML コメントなので Markdown プレビューでは非表示、成果物のレンダリングは clean
